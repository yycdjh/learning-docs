# 链表 —— 移出链表元素 —— leetcode 203

1.leetcode

```javascript
function removeElements(head: ListNode | null, val: number): ListNode | null {
  const data = new ListNode(0, head);
  let cur = data;
  while (cur.next) {
    if (cur.next.val === val) {
      cur.next = cur.next.next;
      continue;
    }
    cur = cur.next;
  }

  return data.next;
}
```
